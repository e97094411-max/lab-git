# Отчет по использованию AI в Лабораторной работе №5

Использованный инструмент: DeepSeek (веб-версия)

Задача: Написать программу на Python для вычисления суммы чисел из файла.

Промпт (запрос):
"Напиши на Python программу, которая считает, вычисляет их сумму и записывает результат. Программа должна корректно обрабатывать ситуацию, если файл input.txt не найден."

Что предложил AI: Предложил код, который выполняет требуемые действия. Также предложил альтернативную версию этого кода только с целыми числами. Привел пример использовуния и выделил 5 особенностей программы.

Доработка: Код был принят без изменений.
